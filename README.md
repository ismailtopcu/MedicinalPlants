# MedicinalPlants
Medicinal Plants projesi N katmanlı mimaride, repository design pattern ile hazırlanmıştır. Projemde .Net 6.0 sürümünü kullanarak, Api Consume işlemlerini yaptım.
# Technologies
- .Net 6.0
- MVC
- .Net Web API
- Entity Framework
- Object-Oriented Programming
